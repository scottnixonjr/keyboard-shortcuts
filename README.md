# Keyboard Shortcuts

⌃  <- Control

⌥  <- Option aka alt

⌘  <- Command

⇧  <- Shift

[Mac Keyboard Shortcuts](https://support.apple.com/en-us/HT201236)

## Shell

**Moving**

^A - beginning of the line

^E - end of the line

^/ - undo


**Delete**

^U - delete everything

^W - delete all left of cursor

^K - delete all right of cursor

`stty all` - show shortcuts and settings.
